# Mr Mwirigi Mathematics Training Hub

A static learning website for Civil Engineering, Building Technology, and Land Survey mathematics trainees.

## Main sections

- Civil Engineering Mathematics
- Building Technology Mathematics
- Land Survey Mathematics
- Assignments
- Revision Materials
- About
- Contact

## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files and folders in this package to the repository.
3. Go to **Settings**.
4. Open **Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and `/root` folder.
7. Save and wait for GitHub Pages to publish the site.

## How to add learning materials

Place files in the relevant folder:

- `materials/civil/`
- `materials/building/`
- `materials/land-survey/`
- `assignments/civil/`
- `assignments/building/`
- `assignments/land-survey/`
- `revision/civil/`
- `revision/building/`
- `revision/land-survey/`

Then update the relevant HTML page to link directly to the file.
