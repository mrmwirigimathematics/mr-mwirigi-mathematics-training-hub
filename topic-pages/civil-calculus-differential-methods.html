Section 1

Rates of change

Gradient and derivative as measures of engineering change.

In civil engineering, this concept supports technical judgement in tasks such as road profiles. The learner must connect the numerical answer to site, drawing, survey, measurement, or data decisions.

Core formula

\[\frac{d}{dx}(ax^n)=anx^{n-1}\]

Worked method

Road profile gradient

For y = 0.02x² + 0.5x + 100, find the gradient at x = 20 m.

Method: identify the given values, write the correct formula, substitute carefully, simplify, attach units, and interpret the result.

Common errors

  • Using a formula before checking what each symbol means.
  • Mixing units such as mm, cm and m.
  • Rounding too early.
  • Giving a final number without technical interpretation.

Practice

  1. Create and solve one original problem involving road profiles.
  2. Explain the formula used and define every symbol.
  3. State the answer with units and one technical interpretation.
