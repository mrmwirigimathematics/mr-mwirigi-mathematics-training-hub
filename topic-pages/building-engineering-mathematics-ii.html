Section 1

Advanced algebra

Equations, functions and construction formulas.

In building technology, this concept supports technical judgement in tasks such as cost optimization. The learner must connect the numerical answer to site, drawing, survey, measurement, or data decisions.

Core formula

\[\frac{dy}{dx}=f'(x)\]

Worked method

Cost optimization

Differentiate a quadratic cost function and find its stationary point.

Method: identify the given values, write the correct formula, substitute carefully, simplify, attach units, and interpret the result.

Common errors

  • Using a formula before checking what each symbol means.
  • Mixing units such as mm, cm and m.
  • Rounding too early.
  • Giving a final number without technical interpretation.

Practice

  1. Create and solve one original problem involving cost optimization.
  2. Explain the formula used and define every symbol.
  3. State the answer with units and one technical interpretation.
