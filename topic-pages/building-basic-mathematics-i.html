Section 1

Number operations

Whole numbers, decimals and calculator discipline.

In building technology, this concept supports technical judgement in tasks such as mortar mixing. The learner must connect the numerical answer to site, drawing, survey, measurement, or data decisions.

Core formula

\[\frac{a}{b}+\frac{c}{d}=\frac{ad+bc}{bd}\]

Worked method

Mortar ratio

A 1:4 mortar mix requires 50 kg cement. Estimate sand required.

Method: identify the given values, write the correct formula, substitute carefully, simplify, attach units, and interpret the result.

Common errors

  • Using a formula before checking what each symbol means.
  • Mixing units such as mm, cm and m.
  • Rounding too early.
  • Giving a final number without technical interpretation.

Practice

  1. Create and solve one original problem involving mortar mixing.
  2. Explain the formula used and define every symbol.
  3. State the answer with units and one technical interpretation.
