Formula summary

Key formulas

The formulas below are rendered using MathJax for professional mathematical display.

Gradient\[\mathrm{Gradient}=\frac{\mathrm{Rise}}{\mathrm{Run}}\]
Departure\[\Delta E=L\sin\theta\]
Latitude\[\Delta N=L\cos\theta\]
Coordinate area\[A=\frac{1}{2}\left|\sum x_iy_{i+1}-\sum y_ix_{i+1}\right|\]
Distance\[d=\sqrt{(\Delta E)^2+(\Delta N)^2}\]

Application areas

  • levelling
  • traverse computation
  • control lines
  • area by coordinates
  • survey closures
Section 1

Coordinate geometry

Distance, gradient and line equations.

In land survey, this concept supports technical judgement in tasks such as levelling. The learner must connect the numerical answer to site, drawing, survey, measurement, or data decisions.

Core formula

\[\mathrm{Gradient}=\frac{\mathrm{Rise}}{\mathrm{Run}}\]

Worked method

Traverse component

Find latitude and departure for a 100 m line at bearing 35°.

Method: identify the given values, write the correct formula, substitute carefully, simplify, attach units, and interpret the result.

Common errors

  • Using a formula before checking what each symbol means.
  • Mixing units such as mm, cm and m.
  • Rounding too early.
  • Giving a final number without technical interpretation.

Practice

  1. Create and solve one original problem involving levelling.
  2. Explain the formula used and define every symbol.
  3. State the answer with units and one technical interpretation.
