Formula summary

Key formulas

The formulas below are rendered using MathJax for professional mathematical display.

Distance from coordinates\[d=\sqrt{(\Delta E)^2+(\Delta N)^2}\]
Bearing angle\[\theta=\tan^{-1}\left(\frac{\Delta E}{\Delta N}\right)\]
Sine\[\sin\theta=\frac{O}{H}\]
Cosine\[\cos\theta=\frac{A}{H}\]
Tangent\[\tan\theta=\frac{O}{A}\]

Section 1

Survey arithmetic

Units, rounding and field-book accuracy.

In land survey, this concept supports technical judgement in tasks such as field distances. The learner must connect the numerical answer to site, drawing, survey, measurement, or data decisions.

Core formula

\[d=\sqrt{(\Delta E)^2+(\Delta N)^2}\]

Worked method

Coordinate distance

A point changes 60 m east and 80 m north. Find distance.

Method: identify the given values, write the correct formula, substitute carefully, simplify, attach units, and interpret the result.

Common errors

  • Using a formula before checking what each symbol means.
  • Mixing units such as mm, cm and m.
  • Rounding too early.
  • Giving a final number without technical interpretation.

Practice

  1. Create and solve one original problem involving field distances.
  2. Explain the formula used and define every symbol.
  3. State the answer with units and one technical interpretation.
